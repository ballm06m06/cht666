# Mei-Chu Hackathon. 2018 -cht team

#### 主要功能：
* 語音辨識-GoogleSpeechToText、意圖判定-Dialogflow、 NLP-Olami
* 魚塭整體資訊
* 水質酸鹼、溶氧量、溫度查詢
* 魚種設定

#### 系統建置：
* python-flask app
* backend: node.js REST API 
* DB: PostgreSQL
